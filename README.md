# Graphics Engine - Deferred Shading
Deferred Shading tecnique implemented by two 4th year students: Victor Chen and Josep Lleal for the Advanced Graphics Programming subject.

## Camera controls
- W: Move forward
- S: Move backward
- D: Move right
- A: Move left
- E: Rotate right
- Q: Rotate left
